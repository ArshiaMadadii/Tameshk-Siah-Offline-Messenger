مجوز استفاده شخصی (PUOL)

سال اجرا: ۲۰۲۶

کپی‌رایت (c) ۲۰۲۶
Arshia Madadi
تمامی حقوق محفوظ است.

این مجوز شامل تمامی نرم‌افزارها، فریمورها، کدهای منبع، مستندات، طراحی‌های سخت‌افزار، فایل‌های PCB، شماتیک‌ها، مدل‌های سه‌بعدی، یادداشت‌های تحقیقاتی، تصاویر و هر نوع ماده دیگری که درون این پروژه قرار دارد یا از آن مشتق شده است (که از این پس «مواد» نامیده می‌شود) می‌باشد.

۱. فقط برای استفاده شخصی
به هر فرد اجازه داده می‌شود مواد را تنها برای استفاده شخصی، خصوصی و غیرتجاری دانلود، مشاهده و اصلاح نماید. استفاده شخصی شامل هیچ‌گونه استفاده سازمانی، دانشگاهی، شرکتی، دولتی یا آموزشی نمی‌شود.

۲. ممنوعیت کامل استفاده تجاری
هرگونه استفاده تجاری بدون اجازه کتبی قبلی از صاحب کپی‌رایت به‌طور کامل ممنوع است. استفاده تجاری شامل، اما محدود به موارد زیر نمی‌باشد:
- فروش مواد
- فروش سخت‌افزار ساخته‌شده بر اساس مواد
- استفاده از مواد در محصول یا سرویس تجاری
- مشاوره یا توسعه مبتنی بر مواد در قبال پول
- صدور مجوز یا واگذاری مجوز
- جمع‌آوری کمک مالی یا توزیع در قبال حمایت مالی

۳. ممنوعیت کامل استفاده آکادمیک و پژوهشی
مواد بدون اجازه کتبی قبلی صاحب کپی‌رایت نباید در فعالیت‌های دانشگاهی، پژوهشی، علمی، دولتی یا آموزشی مورد استفاده قرار گیرد. این شامل موارد زیر می‌شود:
- دروس و آزمایشگاه‌های دانشگاهی
- پایان‌نامه، پروژه کارشناسی، یا رساله
- مقالات علمی یا پژوهشی
- کنفرانس‌ها، پوسترها یا ارائه‌های علمی
- مسابقات دانشجویی یا کارگاه‌ها
- پژوهش‌های دولتی یا نظامی

۴. ممنوعیت کامل انتشار و افشا
هیچ بخشی از مواد بدون اجازه کتبی قبلی صاحب کپی‌رایت نباید منتشر، افشا، نمایش، ارائه یا در رسانه‌های عمومی یا خصوصی استفاده شود. این شامل موارد زیر است:
- ریپوزیتوری‌های عمومی مانند GitHub
- وب‌سایت‌ها یا شبکه‌های اجتماعی
- پلتفرم‌های ویدئویی مانند YouTube
- نشریات چاپی یا دیجیتال
- مستندات فنی یا ارائه‌های عمومی

۵. محدودیت تولید و توزیع سخت‌افزار
ساخت، مونتاژ، توزیع، فروش یا اهدا کردن سخت‌افزار مبتنی بر مواد بدون اجازه کتبی قبلی صاحب کپی‌رایت، چه با هدف تجاری و چه غیرتجاری، ممنوع است.

۶. محدودیت در آثار مشتق‌شده
ایجاد آثار مشتق‌شده تنها برای استفاده شخصی مجاز است و نباید:
- توزیع
- انتشار
- اوپن‌سورس
- صدور مجوز
- فروش
- ارائه عمومی
- استفاده آکادمیک
گردد.

تمام آثار مشتق باید این مجوز و نام صاحب اثر اصلی را حفظ کنند. حذف یا تغییر اطلاعیه‌های کپی‌رایت ممنوع است.

۷. ممنوعیت صدور مجوز مجدد
مواد نباید بدون اجازه کتبی صاحب کپی‌رایت تحت هر نوع مجوز دیگری منتشر یا ترکیب شوند. هرگونه صدور مجوز مجدد یا Sub-license ممنوع است.

۸. عدم ارائه ضمانت
مواد «همان‌گونه که هستند» بدون هیچ ضمانتی ارائه می‌شوند. هیچ‌گونه ضمانت صریح یا ضمنی از جمله کارایی، امنیت یا عدم نقض وجود ندارد.

۹. محدودیت مسئولیت
در هیچ حالتی صاحب اثر مسئول هیچ‌گونه خسارت مستقیم یا غیرمستقیم ناشی از استفاده یا عدم امکان استفاده از مواد نخواهد بود.

۱۰. درخواست مجوز
هر فرد یا سازمانی که قصد:
- انتشار
- پژوهش
- تجاری‌سازی
- تولید
- ارائه عمومی
- توزیع
- آموزش
را دارد باید از صاحب کپی‌رایت اجازه کتبی بگیرد.
